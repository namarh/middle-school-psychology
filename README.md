## 📋 نبذة عن المشروع

عرض تعليمي تفاعلي مخصص للمعلمين لفهم الخصائص النفسية لطلاب المرحلة المتوسطة
(الفئة العمرية 14–16 سنة) وتوظيف هذا الفهم في بيئة الفصل الدراسي.

**المعلم:** يحيى الشمراني  
**المدرسة:** متوسطة مكة المكرمة  
**المادة:** المهارات الرقمية  

---

## ✨ المميزات

- **6 شاشات تفاعلية** تغطي جميع جوانب النمو النفسي
- **ربط مباشر بالفصل الفعلي** — تحليل مبني على بيانات حقيقية
- **تصميم احترافي** يعمل بدون إنترنت (ملف واحد مكتفٍ بذاته)
- **دعم كامل للعربية** مع واجهة RTL
- **خطة تدخل** مبنية على علم النفس التربوي

---

## 📚 محتوى العرض

| الشاشة | المحتوى |
|--------|---------|
| 1️⃣ نظرة عامة | مقدمة الخصائص النفسية للمرحلة |
| 2️⃣ النمو المعرفي | التفكير المجرد، الانتباه، التعلم الاكتشافي |
| 3️⃣ النمو الانفعالي | التقلبات، الحساسية للنقد، الحاجة للاستقلالية |
| 4️⃣ النمو الاجتماعي | تأثير الأقران، الانتماء، التنافس الصحي |
| 5️⃣ الدافعية والتحصيل | محفزات التعلم، قاتلات الدافعية، التدخل المبكر |
| 6️⃣ خطة العمل | 5 إجراءات فورية + جدول تدخل شهري |
